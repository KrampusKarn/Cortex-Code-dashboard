## Workshop Demo Setup — Path A (Mock API + Full Medallion ELT)

**Account:** `sevenpeaks_partner_demo`  
**Tunnel:** ngrok  
**Path:** A — mock API → Bronze → Silver → Gold → App + Documents

---

### Pre-requisites to verify before running

1. `snow` CLI installed and `sevenpeaks_partner_demo` connection configured in `~/.snowflake/connections.toml`
2. `ngrok` installed (`brew install ngrok`) with authtoken configured (`ngrok config add-authtoken <token>`)
3. A stable ngrok domain (either free static domain or paid) — needed for `NGROK_DOMAIN`
4. Python 3.9+ available
5. Cortex enabled on the account + `GRANT DATABASE ROLE SNOWFLAKE.CORTEX_USER TO ROLE ACCOUNTADMIN;`

---

### Execution Steps (ordered)

**Step 1 — Verify connection**
```bash
snow connection test -c sevenpeaks_partner_demo
```

**Step 2 — Run setup SQL** (creates database, warehouses, PUBLIC schema, chat + document tables)
```bash
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -f examples/hris_people/deployed_app/src/00_setup.sql
```

**Step 3 — Start the mock API + ngrok tunnel**
```bash
cd examples/hris_people/deployed_app/mock_api
./serve_eai.sh start --connection sevenpeaks_partner_demo --set-rule
```
This will:
- Create a local `.venv` and install FastAPI/uvicorn/faker
- Boot the API on localhost:8000
- Open the ngrok tunnel on your stable domain
- Set the network rule (`--set-rule` needed on first run)

Requires: `.tunnel.env` file with `NGROK_DOMAIN=<your-stable-domain>` (or pass `--ngrok-domain <host>`)

**Step 4 — Create Bronze/Silver/Gold schemas + EAI + procedures**
```bash
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -f examples/hris_people/deployed_app/src/02_bronze.sql
```

**Step 5 — Ingest from mock API into Bronze**
```bash
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -q \
  "CALL DEMO_EMPLOYEE_APP.BRONZE.SP_INGEST_ALL_BRONZE('https://<your-ngrok-domain>');"
```

**Step 6 — Build Silver** (flatten Bronze VARIANT into typed tables)
```bash
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -f examples/hris_people/deployed_app/src/03_silver.sql
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -q "CALL DEMO_EMPLOYEE_APP.SILVER.SP_BUILD_SILVER();"
```

**Step 7 — Build Gold + Semantic Analyst + Document Ingestion**
```bash
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -f examples/hris_people/deployed_app/src/04_gold.sql
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -f examples/hris_people/deployed_app/src/05_semantic_analyst.sql
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -f examples/hris_people/deployed_app/src/01_document_ingestion.sql
```

**Step 8 — Deploy the Streamlit app**
```bash
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -f examples/hris_people/deployed_app/src/deploy_app.sql
```
(Or if updating an existing app: `cd examples/hris_people/deployed_app/app && snow streamlit deploy --replace -c sevenpeaks_partner_demo`)

**Step 9 — Upload RAG documents + rebuild chunks**
```bash
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -q \
  "PUT 'file://examples/hris_people/deployed_app/docs/*.md' @DEMO_EMPLOYEE_APP.PUBLIC.COMPANY_DOCS AUTO_COMPRESS=FALSE OVERWRITE=TRUE;"
snow sql -c sevenpeaks_partner_demo --role ACCOUNTADMIN -q \
  "ALTER STAGE DEMO_EMPLOYEE_APP.PUBLIC.COMPANY_DOCS REFRESH; CALL DEMO_EMPLOYEE_APP.PUBLIC.SP_REBUILD_DOC_CHUNKS();"
```

**Step 10 — Stop the tunnel when done**
```bash
cd examples/hris_people/deployed_app/mock_api
./serve_eai.sh stop
```
